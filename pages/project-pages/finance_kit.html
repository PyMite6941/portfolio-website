Description


I originally started this project as a fun programming thing to work on for improvement while flying to Tokyo for vacation with my family. However, my AP Computer Science Principles class required a submission of a project that I needed to make and I decided to submit the basic version of this project. Let's just say that this program has turned into so so so much more than what I initially envisioned when I first committed the repo.
This project has taught me so much about developing a program for a commercial and open-source project, especially working with front and back end and how to make sure everything connects. My favorite part was the backend design because the solutions were elegant in my opinion.
