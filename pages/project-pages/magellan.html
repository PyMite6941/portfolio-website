Description


I wanted to build something that actually did something useful, so I made a search engine. Magellan has two parts — a Go spider that crawls websites and stores what it finds in an SQLite database, and a React frontend that lets you search through it. It's named after the explorer because it goes out and finds things. I built the spider first as a standalone tool and then wired up the React side to make it actually usable. It's not Google, but it works and I learned a lot building it.
