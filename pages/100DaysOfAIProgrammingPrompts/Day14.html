Day 14

A Rust Message Encryptor

Unfortunately, you can't run this in the browser here

This program reads input from the terminal, so it must be run locally — it cannot run in the browser.

Run cargo run to start.


Description


A Rust CLI program that takes a message and a key and encrypts the message using the key. A good exercise for learning how Rust handles strings, user input, and basic cryptography logic.

View the source code


use std::io; fn main() { println!("What message should be encrypted?"); let mut message = String::new(); io::stdin().read_line(&mut message).expect("Failed to read"); println!("What is the key to encrypt by?"); let mut key = String::new(); io::stdin().read_line(&mut key).expect("Failed to read"); let mut message_bytes = message.as_bytes(); let mut key_bytes = key.as_bytes(); let mut encrypted: Vec = Vec::new(); let mut i = 0; loop { if i >= message_bytes.len() { break; } let encrypted_bytes = message_bytes[i] ^ key_bytes[i % key_bytes.len()]; encrypted.push(encrypted_bytes); i += 1; } println!("{}",encrypted); }
