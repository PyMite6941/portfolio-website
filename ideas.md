# Ideas — portfolio-website

- Project filter bar: filter cards by technology tag (Python, React, Go, etc.)
- Dark mode toggle: CSS custom properties swap with a `data-theme` attribute
- Animated page transitions: fade-in on load using CSS `@keyframes`
- Contact form: static HTML form that posts to Formspree or EmailJS (no backend needed)
- Blog / articles section: a `pages/articles/` depth-2 page listing short write-ups
- Mobile nav menu: hamburger button that toggles the nav links on small screens
- Add missing projects: Pixel Assistant, Create-Performance-Task, anniversary-project, magellan
- Project detail pages: expand each portfolio card to a full depth-2 page with screenshots and tech notes
- "Live demo" badges on cards that link directly to the running project
- Favicon and Open Graph meta tags for better link previews when shared
